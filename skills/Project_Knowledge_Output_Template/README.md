# Project Knowledge Output Template

This package contains the recommended output format for the Project Bootstrap & Codebase Indexing skill.

Files:
- knowledge-output-template.md : Human-readable knowledge base template
- knowledge.json (to be generated by the skill): Machine-readable context for downstream AI agents

Recommended pipeline:

Project Bootstrap
    ↓
knowledge.md + knowledge.json
    ↓
Requirement Analyst
    ↓
Test Case Generator
    ↓
Test Case Reviewer
    ↓
Automation Generator
    ↓
Automation Reviewer
    ↓
Execution Analyzer
